# C Hesap Makinesi Projesi

Bu proje, Yıldız Teknik Üniversitesi Matematik Mühendisliği bölümündeki çalışmalarım kapsamında geliştirdiğim temel bir hesap makinesidir.

## Özellikler
- **Dört İşlem:** Toplama, çıkarma, çarpma ve bölme işlemlerini destekler.
- **if-else Yapısı:** Karar mekanizmaları kullanılarak mantıksal akış sağlanmıştır.
- **Hata Kontrolü:** Bölme işleminde sıfıra bölünme hatası (undefined) kontrol edilir.

## Nasıl Çalıştırılır?
1. Herhangi bir C derleyicisi (GCC vb.) ile derleyin.
2. Terminale şu komutu yazın:
   ```bash
   gcc hesap.c -o hesap
   ./hesap
